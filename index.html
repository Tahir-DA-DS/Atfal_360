360 API

  • Get all atfal details https://atfal-360.onrender.com/v1/atfals
  • Filter atfal details https://atfal-360.onrender.com/v1/atfalsfilter?State=FCT here filtering by state if you want to filter by Age , change State=FCT to Age=the age you want to filter by
  • Get total atfal https://atfal-360.onrender.com/v1/totalAtfal
  • Get total atfal by category https://atfal-360.onrender.com/v1/dataStat
  • Download atfal data use https://atfal-360.onrender.com/v1/download
  • Download atfal data base on filtered data https://atfal-360.onrender.com/v1/download?Age=7 this download all atfals who are 7 years old
